HTML

Elemen semantik adalah tag-tag HTML yang mempunyai makna dan tujuan beradasarkan elemen itu sendiri.
Misalnya tag header, tag ini digunakan untuk bagian atas dari web. Jangan gunakan tag ini di bagian paling
bawah, karena setiap tag sudah punya tujuan masing-masing. Elemen semantik ini baru mulai ditambahkan
pada HTML 5, yang biasanya digunakan untuk membuat layout ataupun untuk membuat elemen lain.
